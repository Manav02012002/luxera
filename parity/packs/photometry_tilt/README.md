# photometry_tilt

Parity pack for IES TILT behavior and transform ordering.

Scenes:
- `downward_tilt`: TILT=INCLUDE with local downward-aim baseline.
- `sideways_tilt`: TILT=FILE with rotated luminaire orientation.
- `combined_tilt_aim`: TILT active with non-zero luminaire `tilt_deg` and explicit aiming rotation.
